# DevBox - 开发者在线工具箱

8个实用开发工具，数据全部本地处理，无需服务器。

## 工具列表

- **JSON 格式化** — 格式化、压缩与语法验证
- **时间戳转换** — Unix 时间戳与日期双向转换  
- **URL 编解码** — 参数编码与解码，支持中文
- **Base64 编解码** — 文本与 Base64 互转，UTF-8 支持
- **Markdown 预览** — 实时渲染 GFM 格式文档
- **正则测试器** — 在线测试、匹配与替换
- **颜色转换** — HEX / RGB / HSL 互转
- **文本处理** — 去重、排序、大小写、反转

## 使用

直接在浏览器中打开 `index.html`，或部署到任意静态服务器。

## 技术

- 纯 HTML/CSS/JS，零依赖（除 Markdown 渲染使用 marked.js CDN）
- 所有数据在浏览器本地处理，不上传服务器
- 支持深色/浅色主题切换
